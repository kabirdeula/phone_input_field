# Changelog

## [0.0.1] - 2025-04-09

### Added
- Initial release of international_phone_input package
- International phone number input field widget
- Country selection dialog with search functionality
- Support for 20+ countries with flag emojis and dial codes
- Clear/reset button in phone input field
- Custom prefix icon support
- Country and phone number change callbacks
- Example app demonstrating usage

### Features
- `InternationalPhoneInputField` - Main widget for phone input
- `Country` - Model for country data
- `CountryPickerDialog` - Dialog for country selection
- Pre-loaded countries data (expandable)

### Notes
- This is an early release with basic functionality
- Phone number validation will be added in future releases
- More advanced formatting options coming soon
